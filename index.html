Actividad 1 - Desarrollo Web

Investigación de Tópicos

Este tema es fundamental para comprender cómo se construyen las páginas web. A continuación, abordamos los siguientes subtemas:

  • Orígenes de la Web: Exploraremos cómo surgió la World Wide Web, desde sus inicios como un proyecto de investigación hasta convertirse en la herramienta de comunicación global que conocemos hoy.
  • Web 2.0: Analizaremos la evolución de la web hacia una plataforma más interactiva y colaborativa, caracterizada por la generación de contenido por parte de los usuarios.
  • Web 3.0: Exploraremos las tendencias actuales y futuras de la web, como la descentralización, la inteligencia artificial y el uso de blockchain.
  • La W3C y los estándares: Comprenderemos el papel del World Wide Web Consortium en la definición de los estándares que rigen el desarrollo web.
  • Lenguajes de etiquetas:
    • SGML: Exploraremos el lenguaje de marcado genérico que sirvió como base para otros lenguajes de marcado.
    • XML: Analizaremos el lenguaje extensible de marcado utilizado para almacenar y transportar datos estructurados.
    • HTML: Estudiaremos el lenguaje de marcado de hipertexto utilizado para crear la estructura y contenido de las páginas web.
    • XHTML: Exploraremos la versión extensible de HTML que combina las características de HTML y XML.
    • HTML5: Analizaremos la última versión de HTML, que incorpora nuevas características y mejora la experiencia del usuario.
  • Otros lenguajes relacionados al desarrollo web:
    • CSS/CSS3: Estudiaremos los lenguajes utilizados para dar estilo y formato a las páginas web.
    • JavaScript: Exploraremos el lenguaje de programación utilizado para crear interacciones dinámicas en las páginas web.
    • Ajax: Analizaremos la técnica que permite actualizar partes de una página web sin recargarla por completo.
    • Tecnologías y lenguajes del lado del servidor: Exploraremos lenguajes como PHP, Python y Node.js.
  • Navegadores: Analizaremos los diferentes navegadores web y su funcionamiento.
  • Motores renderizados: Exploraremos cómo los navegadores interpretan el código HTML, CSS y JavaScript para mostrar las páginas web.
  • Herramientas de desarrollo:
    • Software WYSIWYG: Estudiaremos herramientas que permiten crear páginas web de forma visual, sin escribir código.
    • Otras alternativas: Figma, InVision, Zeplin, GitHub.
  • Inspección de código desde el navegador: Exploraremos las herramientas que permiten inspeccionar y modificar el código HTML, CSS y JavaScript en tiempo real.

Metodología de Investigación

Para cada subtema, realizaré una investigación exhaustiva utilizando diversas fuentes:

  • Libros y manuales especializados.
  • Artículos académicos y publicaciones.
  • Documentación oficial.
  • Tutoriales en línea (YouTube, Udemy, Coursera).
  • Foros y comunidades en línea.

Reglas para Escribir HTML

  • Cada elemento debe tener una etiqueta de apertura y cierre.
  • Las etiquetas deben estar correctamente anidadas.
  • Los atributos proporcionan información adicional sobre los elementos.
  • Usa minúsculas para etiquetas y atributos.
